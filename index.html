My definition of Git is: Git is a free to use version control system created by Linux which makes collaboration

Define version control.
Version control system are the sytems used by developers while collaborating remotely on a project. It makes ie easier for them to Write code,change code and debug code remotely and push it to the internet.

Processes involved in commiting a project to github repository.
  1. create a github account
  2. create a github repository.(repository is just like folder but would be on the web)
  3. Write your code with preferable text editor.
  4. using the command line on windows or terminal on MAC, initialize the git
  5. use the git add command after every file modification
  6. Use the "git commit" when you're ready to push your file to the internet. And lastly,
  7. use the "git push origin main" main command to push the file to yourgithub.
